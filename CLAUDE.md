## Project Overview

Courtesy Clerk is a Godot 4.4 game project using the Forward Plus renderer.

## Development Environment

- **Engine**: Godot 4.4
- **Renderer**: Forward Plus
- **Language**: GDScript

## Running the Project

Open the project in Godot Editor:
```
godot --editor --path .
```

Run the game from command line:
```
godot --path .
```

## Godot Conventions

- Scene files use `.tscn` extension
- Script files use `.gd` extension (GDScript)
- Resources use `.tres` extension
- The `.godot/` directory contains generated files and should not be edited manually
- `project.godot` is the main project configuration file
